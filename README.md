# Array Algorithms in C

This repository contains C programs that demonstrate common array algorithms and operations. These programs are useful for practicing array manipulation and improving problem-solving skills in C.

## Programs Included

- Triplets (`triplets.c`)
- Second Largest Element (`second-largest.c`)
- Reverse Copy of an Array (`reverse-copy.c`)
- Reverse an Array (`reverse.c`)
- Rotate Array (`rotate-arrays.c`)

## Features

- Array traversal
- Array reversal
- Array rotation
- Finding the second largest element
- Triplet-based array operations

## Technologies Used

- C Programming Language
- GCC Compiler
- Visual Studio Code / Code::Blocks

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/Ayan-nawaz/array-algorithms-c.git
   ```

2. Navigate to the project directory:
   ```bash
   cd array-algorithms-c
   ```

3. Compile a program:
   ```bash
   gcc filename.c -o output
   ```

4. Run the executable:
   ```bash
   ./output
   ```

## Purpose

This repository is created for learning and practicing array-based algorithms in C programming.
